# Start here

`AGENTS.md` (imported above) is the shared engineering rulebook for this repo —
the same file Claude loads via `CLAUDE.md`. Both agents are held to it.

Before writing any code, also read **`docs/HANDOFF.md`** — it records what is
already built, the current branch state, and the agreed next tasks.

If you intend to deviate from anything in `AGENTS.md`, say so explicitly and why,
rather than silently doing it differently. If you change a shared convention,
update `AGENTS.md` in the same commit so the other agent picks it up.
